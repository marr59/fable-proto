# Fable Proto

Браузерная 2.5D изометрическая RPG в стиле феодальной Азии с тёмным фэнтези.
Ключевая механика — карма (±100), визуально меняющая персонажа и мир.

## Стек и ограничения
- Three.js 0.168, чистый браузер: ES-модули + ImportMap → CDN jsDelivr. БЕЗ npm/bundler и без шага сборки.
- Пост-обработка: UnrealBloomPass (EffectComposer). Модели: GLTFLoader (не используется для персонажей).
- Запуск: `python3 -m http.server 8000` в папке проекта.

## Текущее состояние
- Сцена: ночное небо (1500 звёзд, луна), красные тории, каменная дорожка, фонари-торо с мерцанием, 14-blob сакура + мёртвые деревья (IcosahedronGeometry), 3 анимированных костра, 2 пагоды со светящимися окнами (bloom), разрушенные стены, колодец (вода + BackSide inner).
- Юниты (RTS): 4 класса — Крестьянин, Лучник, Копейщик, Алхимик. Риггованные модели: SkeletonUtils.clone(Xbot.glb) с авто-масштабом (Box3 → 1.7 ед высота) и перекраской по классу (CLASS_COLOR). Класс = цвет + оружие в руке: лучник → лук (TorusGeometry), копейщик → копьё (shaft+blade+tassel), алхимик → посох с орбой; крестьянин без оружия. Скелетная анимация: Idle/Walk кроссфейд 0.2с через AnimationMixer. Xbot.glb грузится с raw.githubusercontent.com/mrdoob/three.js/r168.
- Гильдии: 3 здания (лучников 0x39c24a, копейщиков 0xd0402e, алхимиков 0x9a3ad0) с баннером и эмблемой.
- Обучение: крестьянин идёт к входу гильдии, тренируется 3.5 сек → трансформируется в воина. Прогресс-бар в HUD.
- Выделение: рамка мышью (drag) или клавиша A/Ф — выбрать всех; клик по юниту — выбрать одного. Групповые приказы (move/train) работают для всех выделенных.
- Переобучение: любой юнит (любого класса) → любая гильдия → трансформация в новый класс.
- Click-to-move: клик по земле — группа движется; по гильдии — обучаться.
- Камера: lerp-следование за центром выделения; при снятом выделении — свободная прокрутка (cursor к краю экрана, десктоп, скорость 16 ед/с).
- Суставная анимация: отдельные Group для ног/рук/торса; шаг (swing) при движении, idle-дыхание (scale торса + покачивание оружия).
- Коллизии: скользящий барьер (r=2.2 пагоды, r=2.3 гильдии), clampOut при движении.
- Ввод: pointerdown + touchAction:none — работает на iPhone/iPad.
- Карма ±100: >+25 → золотые частицы + bloom; <−25 → фиолетовые + тёмное свечение. PointLight следует за выбранным юнитом.
- HUD (HTML/CSS): шрифт Cinzel, маска они, HP-бар, значение кармы, метка класса юнита, прогресс-бар обучения, подсказка внизу экрана.
